# DEG Function

Converts the value of an angle from radians to degrees.
  
## Syntax

DEG(** *angle* ** ) 
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |Required  <br/> |**String** <br/> |The value of the angle in radians.  <br/> |
   
## Example

DEG(PI( )/4) 
  
Returns 45
