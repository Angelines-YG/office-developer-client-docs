# DEG Function

Converts the value of an angle from radians to degrees.
  
## Syntax

DEG(***angle*** )
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| *angle* <br/> |Required  <br/> |**String** <br/> |The value of the angle in radians. |

## Example

DEG(PI( )/4)
  
Returns 45
